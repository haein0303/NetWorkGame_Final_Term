{
대기화면
 -> ? (삭제)

메인 로비
 -게임 시작을 누르면 모드선택으로 넘어감(삭제)
 -게임 종료를 누르면 게임 끝(삭제)
 -설명을 누르면?(삭제)


모드 선택
 -> ?(미정인 상태 유지)

캐릭터 선택
 -캐릭터 셋을 보여주고 선택하게 함
 -선택된 캐릭터의 스탠딩 이미지가 나타남
 -선택된 캐릭터의 테두리를 굵게 그려서 선택됬다는걸 알려줌
 -레디 버튼을 누르면 다른 애니메이션(승리)이 나오며 캐릭터를 변경할 수 없음
 -둘 다 레디를 누르면 즉시 시작

인게임
 -시작할 때 3초 카운트(화면 전환 이미지를 출력)
 -타이머 표시 (99~)
 -ESC키 누르면 일시정지되며 팝업 UI가 뜸
   -팝업 UI의 RESUME은 게임 재개, RESTART는 재시작, EXIT는 나가기
// -양쪽에 서로 다른 게임 화면을 보여줌
// -둘이 가까워지면 서로 보임
 -미니맵 출력(반투명  ㅇㅇ)
 -캐릭터별 스킬
 -공격 키를 누르면 공격 공격에 맞은 적은 금화를 떨어뜨리거나 기절
 -다중 키 입력을 받음 ( ㅇㅇ)
 -가속 타일에 올라가면 속도 빨라짐
 -벽 타일은 이동 불가능
 -금화를 들면 미니맵에 표시, 시간이 종료되었을 때 들고있는 사람이 승리
 -없으면 5초이상 금화를 들고있으면 승리
 -게임이 끝나면 승패를 화면에 표시

결과창
 -승/패 표시
 -시간이 지나면 자동으로 대기화면으로 넘어감
 -버튼을 눌러도 넘어감
}
